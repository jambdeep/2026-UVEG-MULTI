# Paso 3: La Interfaz de Usuario (HTML)

## 📝 ¿Qué hace esta sección?
Introduce la estructura visual interactiva de nuestra aplicación. Aquí implementarás el moderno flujo de control nativo de Angular para iterar listas y renderizar bloques condicionales.

## 💡 Conceptos clave
* **`@for` y `@empty`:** Ciclo moderno para dibujar dinámicamente cada entrega o mostrar un aviso amigable si no quedan rutas asignadas.
* **`@if`:** Renderizado condicional para alertar sobre errores de validación.
* **Pipes:** Uso de herramientas de transformación visual directa en la vista (como `uppercase`).
