# 🛵 Mi Primera App de Reparto con Angular

¡Bienvenidos! Este repositorio es el material de apoyo para nuestra sesión en vivo online. El objetivo es construir esta aplicación durante la sesión buscando priorizar el tiempo de una hora. 

## 📊 ¿Qué aprenderás con este proyecto?
Al completar los pasos durante la clase, habrás experimentado con los pilares fundamentales de la evaluación:
1. **TypeScript:** Uso de interfaces para validar la estructura de la información.
2. **Directivas y Renderizado:** El nuevo flujo de control dinámico (`@for`, `@if`) y reactividad en clases (`[class.completada]`).
3. **Formularios:** Validaciones reactivas en tiempo real
4. **Responsividad:** Comportamiento multiplataforma adaptable a móviles y escritorios mediante CSS

## 🗺️ Guía de Navegación por Pasos
Sigue las carpetas en orden cronológico para armar tu aplicación[cite: 1]:
* **`paso-1-proyecto-base/`:** Creación y despliegue del cascarón de la app
* **`paso-2-logica/`:** Modelado de datos y cerebro del componente (`app.component.ts`)
* **`paso-3-interfaz/`:** Estructura visual interactiva (`app.component.html`)
* **`paso-4-estilos/`:** Diseño adaptativo y responsivo (`app.component.css`)
